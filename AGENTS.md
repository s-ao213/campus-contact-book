# Review Guidelines

このプロジェクトは、架空の連絡先を管理する Next.js 製の電話帳アプリケーションです。

レビューでは、以下を重点的に確認してください。

- 実際に不具合につながる論理的なバグ
- 認証と認可
- ハードコードされた認証情報
- 入力値検証
- 電話番号とメールアドレスの扱い
- CSV 出力の安全性
- TypeScript の型安全性
- React および Next.js の適切な使用
- エラー処理
- 可読性と保守性
- アクセシビリティ
- テスト不足
- README やドキュメントの不足

単なる好みや細かなスタイル差ではなく、実際に修正する価値がある問題を優先してください。

レビューコメントには、可能な範囲で以下を含めてください。

1. 問題の内容
2. 問題が発生する条件
3. 想定される影響
4. 具体的な修正案
