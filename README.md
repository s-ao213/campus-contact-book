# Campus Contact Book

研究室や学校内で使う連絡先帳を想定した、Next.js 製のデモアプリです。

## 起動方法

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:3000` を開くと、架空の連絡先一覧を確認できます。

## main ブランチの内容

- 架空の連絡先一覧
- 名前、電話番号、メールアドレス、所属の表示
- Next.js App Router と TypeScript の基本設定
- ESLint の基本設定

このブランチは PR レビュー実演の土台です。連絡先の登録、編集、削除などは feature ブランチで追加します。
